Durch den Erfolg der Microservices entstehen immer mehr zugehörige Frameworks. So wird mit dieser Anwendung eine Bewertungsmethode gestellt, um die Microservice Frameworks anhand einer gegebenen Architektur zu bewerten und miteinander zu vergleichen. Sie wird dabei stark an die Bewertung von Softwarearchitekturen angelehnt, da ein Framework einen erheblichen Einfluss darauf hat.
